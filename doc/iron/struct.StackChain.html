The default Chain used by Iron. StackChain runs each Request through all Middleware in its stack.

When it hits Middleware which returns Unwind, it passes the Request back up through all Middleware it has hit so far.

If no Middleware return Unwind to indicate that they handled the request, then a 404 is automatically returned.
